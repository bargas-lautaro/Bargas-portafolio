Capas avanzadas
LINE
Crea segmentos de línea recta entre puntos
Pto1 → Pto2 o @dist < ángulo
CIRCLE
Crea un círculo con centro y radio
Centro → Radio o @diámetro
RECTANGLE
Crea un rectángulo a partir de dos esquinas opuestas
RECTANGLE → clic en primera esquina → clic en segunda esquina
OFFSET
Crea líneas paralelas a una distancia determinada
OFFSET → ingresar distancia → seleccionar objeto → lado
TRIM
Recorta objetos que se cruzan con otros
TRIM → seleccionar límites → Enter → seleccionar líneas a recortar
EXTEND
Extiende objetos hasta que toquen otros objetos
EXTEND → seleccionar límites → Enter → seleccionar objetos a extender
FILLET
Redondea la unión entre dos líneas con un radio dado
FILLET → R → ingresar radio → seleccionar dos líneas
CHAMFER
Crea un chaflán (bisel) entre dos líneas
CHAMFER → D → ingresar distancias → seleccionar líneas
DIMLINEAR
Cota lineal entre dos puntos horizontales o verticales
DIM → Linear → seleccionar puntos
HATCH
Rellena un área cerrada con un patrón o sombreado
HATCH → seleccionar región → elegir patrón
TEXT
Inserta texto de una sola línea
TEXT → punto → altura → ángulo → texto
MTEXT
Inserta texto en párrafos con formato
MTEXT → dibujar caja → escribir texto
DIM
Cotas automáticas según tipo de objeto
DIM → clic en objeto → ubicar cota
LEADER
Crea flechas con texto explicativo
LEADER → clics → escribir texto
LAYER
Gestión de capas: visibilidad, color, tipo de línea
LAYER → administrador de capas
MATCHPROP
Copia propiedades de un objeto a otro
MATCHPROP → origen → destino
PROPERTIES
Muestra propiedades detalladas del objeto
PROPERTIES → seleccionar objeto
BLOCK
Crea un bloque reutilizable
BLOCK → nombre → objetos → base
INSERT
Inserta un bloque existente
INSERT → elegir bloque → ubicar
ATTEDIT
Edita atributos de un bloque insertado
ATTEDIT → seleccionar atributo → modificar
LAYOUT
Maneja espacios de presentación para impresión
LAYOUT → crear/renombrar presentación
PLOT
Imprime o exporta el dibujo como PDF
PLOT → configurar hoja → imprimir
VIEW
Guarda y restaura vistas personalizadas
VIEW → guardar posición y nombre
DIST
Mide distancia entre dos puntos
DIST → punto 1 → punto 2
AREA
Calcula área y perímetro de una figura
AREA → puntos o región cerrada
ID
Muestra coordenadas exactas de un punto
ID → clic en punto
OSNAP
Activa referencias precisas como extremos, centros o intersecciones
F3 o OSNAP → seleccionar modos de referencia
ORTHO
Restringe el cursor a líneas horizontales o verticales
F8 para activar/desactivar
POLAR
Muestra líneas guía en ángulos definidos (por defecto cada 90°)
F10 para activar/desactivar
DSETTINGS
Abre la ventana de ajustes de dibujo (rejilla, snap, polar, etc.)
DSETTINGS → configurar preferencias
UNITS
Configura unidades de medida del dibujo
UNITS → elegir tipo, precisión y sistema
DONUT
Dibuja un anillo con radio interior y exterior
DONUT → radio int. → radio ext. → centro
SPLINE
Dibuja una curva suave a través de puntos
SPLINE → clics en puntos → Enter
ELLIPSE
Crea una elipse especificando ejes mayor y menor
ELLIPSE → eje mayor → eje menor
REVCLOUD
Crea una nube de revisión alrededor de elementos
REVCLOUD → dibujar contorno
XLINE
Crea líneas infinitas para referencia
XLINE → punto → dirección
MVIEW
Crea ventanas gráficas en espacio presentación
MVIEW → dibujar ventana sobre layout
CHSPACE
Mueve objetos entre espacio modelo y presentación
CHSPACE → seleccionar objeto → destino
MVSETUP
Asistente para configurar hojas de presentación
MVSETUP → configurar vista, escala y layout
LTSCALE
Escala de los tipos de línea del dibujo
LTSCALE → valor → REGEN
REGENALL
Regenera todo el dibujo para actualizar visualización
REGENALL → Enter
EXTRUDE
Convierte objetos 2D en sólidos 3D
EXTRUDE → seleccionar perfil → altura
PRESSPULL
Crea sólidos al presionar o extraer áreas cerradas
PRESSPULL → seleccionar área → altura
UCS
Establece un sistema de coordenadas personalizado
UCS → opciones: origen, objeto, cara, vista
TOOLPALETTES
Abre el panel de herramientas con bloques y comandos
TOOLPALETTES → seleccionar y usar herramientas
ALIAS
Define atajos personalizados en el archivo acad.pgp
Editar acad.pgp → agregar alias
CUI
Personaliza menús, comandos y teclas rápidas
CUI → configurar interfaz
LAYISO
Aísla la capa seleccionada ocultando las demás
LAYISO → seleccionar capa → confirmar
LAYUNISO
Restaura visibilidad de capas tras usar LAYISO
LAYUNISO → Enter
LAYERP
Deshace la última modificación en capas
LAYERP → restaurar estado anterior
PAGESETUP
Configura márgenes, hoja y escala para impresión
PAGESETUP → seleccionar layout → ajustar
PLOTSTYLE
Controla color, grosor y estilo de línea al imprimir
Asignar CTB o STB → desde impresión
PUBLISH
Imprime en lote varias hojas o layouts
PUBLISH → agregar hojas → configurar → publicar
GEOMCONSTRAINT
Aplica restricciones geométricas (paralelo, perpendicular, etc.)
GEOMCONSTRAINT → seleccionar objetos → tipo de restricción
DIMCONSTRAINT
Restringe medidas como distancias o ángulos fijos
DIMCONSTRAINT → seleccionar objeto → ingresar valor
ISOPLANE
Cambia entre planos isométricos izquierdo, derecho o superior
ISOPLANE → usar tecla F5 o ciclo automático
BATTMAN
Administrador de atributos de bloques
BATTMAN → seleccionar bloque → editar atributos
EATTEDIT
Edita atributos de un bloque directamente
EATTEDIT → seleccionar bloque → editar valores
XCLIP
Recorta la visualización de una referencia externa
XCLIP → seleccionar referencia → definir recorte
REFEDIT
Edita una referencia externa sin abrir otro archivo
REFEDIT → seleccionar bloque o Xref
REFCLOSE
Cierra la edición de referencia y guarda los cambios
REFCLOSE → confirmar
DBLIST
Lista todos los objetos del dibujo actual
DBLIST → muestra datos en texto
LIST
Muestra propiedades detalladas de objetos seleccionados
LIST → seleccionar objeto
DWGPROPS
Permite ingresar propiedades del dibujo (autor, título, etc.)
DWGPROPS → editar campos
EXPORT
Exporta el dibujo a otros formatos (PDF, DXF, STL...)
EXPORT → elegir formato → guardar
DWGCONVERT
Convierte archivos DWG a versiones anteriores
DWGCONVERT → seleccionar archivos → elegir versión
RECOVER
Abre y repara un archivo DWG dañado
RECOVER → elegir archivo → verificar correcciones
MOVE
Mueve objetos de un punto a otro
MOVE → seleccionar objetos → Enter → clic origen → clic destino
COPY
Copia objetos a nuevas ubicaciones
COPY → seleccionar objetos → Enter → origen → destino
MIRROR
Crea una copia simétrica respecto a un eje
MIRROR → seleccionar objetos → Enter → eje
ARRAY
Genera copias múltiples en forma rectangular, polar o de camino
ARRAY → elegir tipo → definir cantidad y espaciado
SCALE
Cambia el tamaño de los objetos en proporción
SCALE → seleccionar → Enter → punto base → factor
ROTATE
Rota objetos en torno a un punto base
ROTATE → seleccionar objetos → punto base → ángulo
ZOOM
Acerca o aleja la vista del dibujo
ZOOM → Extents o rueda del mouse
PAN
Desplaza la vista sin cambiar el zoom
PAN o presionar botón central del mouse y arrastrar
OSNAP
Activa referencias precisas como extremos, centros, intersecciones
F3 → o escribir OSNAP → elegir modos
GRID / SNAP
Muestra una rejilla y ajusta los puntos a una grilla
F7 para GRID / F9 para SNAP
