Límite de una función
Asintotas

Las asíntotas son rectas que una función se acerca pero nunca toca. Existen tres tipos principales:

Asíntotas horizontales: Describen el comportamiento de la función cuando x tiende a infinito o menos infinito.
Asíntotas verticales: Ocurren cuando la función tiende a infinito en un valor de x.

Asíntotas oblicuas: Se presentan cuando el grado del numerador es mayor al del denominador en funciones racionales.
Estas ayudan a describir la forma general de una gráfica.

Límites laterales
los límites laterales deben existir y ser iguales
En caso de que esos límites coincidan dan como resultado un límite finito
Se usa para evaluar la continuidad

Los límites laterales de una función describen el comportamiento de la misma cuando la variable se acerca a un valor específico desde la izquierda (−) o desde la derecha (+). Se escriben como:
lim(x→a⁻)f(x) y lim(x→a⁺)f(x).
Si ambos coinciden, existe el límite en x=a.

propiedades
propiedades

Las propiedades de los límites permiten operar con ellos de forma algebraica. Algunas propiedades clave son:
- El límite de una suma es la suma de los límites.
- El límite de un producto es el producto de los límites.
- El límite de una constante por una función es la constante por el límite.
- Si el límite del denominador es distinto de cero, se puede dividir los límites.
Estas propiedades son fundamentales para resolver límites de funciones más complejas.

Casos especiales

Los casos especiales de límites se presentan cuando ocurren situaciones como:
- Indeterminaciones (0/0, ∞/∞, etc.)
- Funciones definidas por partes
- Funciones con asíntotas verticales
- Límites infinitos
En estos casos, se deben aplicar técnicas como factoreo, racionalización, uso de conjugadas o análisis lateral para resolverlos correctamente.

Una función f(x) es continua en x = a si se cumple que \( \lim_{x \to a} f(x) = f(a) \), es decir, el límite existe, coincide con el valor de la función y no hay saltos, huecos ni asíntotas.
Continuidad
El límite de una función es el valor al que se aproxima la función cuando la variable independiente se acerca a un punto específico.
Valor que se aproxima una función cuando la variable independiente se acerca a un número real
Un límite infinito describe el comportamiento de una función cuando sus valores crecen indefinidamente de forma positiva o negativa a medida que la variable independiente se aproxima a un número real o al infinito.
