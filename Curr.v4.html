Administrador de Sistemas

Ejemplos de trabajos que realizo

Configuración de servidores Linux

Instalación, configuración y optimización de servidores Linux adaptados a distintos entornos: caseros, empresariales, escolares y laboratorios. Incluye la selección de distribuciones, ajuste de servicios (web, archivos, correo, bases de datos), configuración de seguridad, usuarios y permisos, automatización de tareas y respaldo. Se realiza tanto en hardware físico como en máquinas virtuales, asegurando estabilidad, rendimiento y facilidad de administración según las necesidades del cliente.

Solución de problemas de conectividad

Diagnóstico y resolución de fallos de conexión en redes cableadas e inalámbricas. Se analizan configuraciones IP, DNS, puerta de enlace, conflictos de red, interferencias en Wi-Fi, problemas de hardware y errores de software. Se utilizan herramientas como ping, traceroute, ipconfig/ifconfig, netstat, Wireshark y escáneres de red. Se aplican soluciones como asignación de direcciones estáticas, reinicio de servicios de red, cambio de canales, reemplazo de componentes defectuosos, y documentación del incidente con recomendaciones preventivas.

Despliegue de entornos de desarrollo

Instalación y configuración de entornos de desarrollo adaptados a distintos lenguajes y frameworks (Python, Node.js, PHP, Java, entre otros). Se implementan servidores locales o virtuales con las herramientas necesarias (editores, compiladores, servidores locales, bases de datos). Se utilizan soluciones como Docker, Vagrant o entornos virtualizados con Visual Studio Code, Git, pip/npm, entornos con aislamiento de dependencias y gestión de versiones. También se automatizan procesos de instalación y documentación para facilitar la reproducción del entorno por distintos equipos de trabajo.

Configuración de servidores web

Implementación y personalización de servidores web como Apache, Nginx y Lighttpd. Se configura alojamiento de sitios estáticos y dinámicos, redirecciones, certificados SSL, balanceo de carga, reglas de seguridad, autenticación de usuarios, compresión, caching y optimización del tiempo de carga. Se integran con PHP, Python, Node.js y bases de datos, asegurando disponibilidad, rendimiento y compatibilidad con estándares modernos. También se implementan herramientas de monitoreo y alertas para mantener la estabilidad del servicio.

Gestión de usuarios y permisos

Administración de cuentas de usuario, grupos y permisos para garantizar la seguridad y el acceso adecuado en sistemas Linux, Windows y plataformas web. Se gestiona el control de acceso tanto en entornos empresariales como en páginas y servicios, asegurando que cada usuario tenga los privilegios necesarios y protegiendo la integridad de los sistemas.

Implementación de políticas de seguridad

Implementación de políticas de seguridad en sistemas Linux mediante herramientas como iptables y firewalld para control de tráfico, SELinux y AppArmor para restricción de permisos, auditd para auditoría de eventos, y fail2ban para protección contra accesos no autorizados. Se configuran reglas, monitoreo y alertas para prevenir amenazas internas y externas, asegurando la integridad y confidencialidad de los datos.

Configuración de VPN

Configuración y optimización de redes privadas virtuales seguras para acceso remoto y protección de datos. Se utilizan herramientas como OpenVPN, WireGuard y StrongSwan. Se asegura el cifrado de extremo a extremo, autenticación por certificados, implementación de túneles divididos y conexión a redes corporativas desde múltiples dispositivos con alta compatibilidad. Se configuran también DNS internos, scripts de reconexión automática y políticas de control de acceso, garantizando privacidad y rendimiento.

Monitoreo de redes

Supervisión constante del tráfico de red mediante herramientas como Wireshark, ntopng, Zabbix y Nagios. Detección de anomalías, latencias y posibles amenazas. Se configuran alertas automatizadas, generación de reportes periódicos y visualización gráfica del estado de enlaces y dispositivos. Permite prevenir fallos antes de que impacten en los usuarios finales y facilita la toma de decisiones para escalabilidad de red.

Soporte técnico avanzado

Resolución de incidentes complejos en sistemas operativos, redes y hardware. Diagnóstico detallado de fallos, recuperación ante errores graves, instalación de software crítico y asistencia remota o presencial. Se abordan problemas relacionados con rendimiento, conectividad, mal funcionamiento de servicios y errores de configuración, ofreciendo soluciones documentadas y adaptadas al entorno específico del cliente.

Automatización con scripts

Desarrollo de scripts personalizados en Bash, Python y PowerShell para automatizar tareas administrativas, copias de seguridad, limpieza de logs, generación de informes, instalación de software, monitoreo de procesos y despliegue de sistemas. Mejora de eficiencia y reducción de errores humanos. También se integran con cron y systemd para tareas programadas y eventos críticos del sistema.

Gestión de backups

Diseño, implementación y mantenimiento de sistemas de respaldo de datos tanto locales como en la nube. Se configuran copias automáticas, restauración ante fallos, pruebas periódicas de recuperación y políticas de retención. Se utilizan herramientas como rsync, Bacula, Veeam, y soluciones cloud (Google Drive, Dropbox, AWS S3) para asegurar la continuidad del negocio y la protección ante pérdida de información por errores humanos, ataques o desastres. Además, se documentan los procedimientos de recuperación y se automatiza la verificación de copias con pruebas de restauración.

Implementación de sistemas de virtualización

Instalación y configuración de plataformas de virtualización como VMware, VirtualBox, Proxmox y KVM. Se crean máquinas virtuales para optimizar recursos, facilitar pruebas, aislar servicios y reducir costos de hardware. Incluye la gestión de snapshots, migración de máquinas, balanceo de carga y administración centralizada. Se aplican configuraciones avanzadas de red virtual, almacenamiento compartido y alta disponibilidad para entornos empresariales y educativos.

Optimización de rendimiento de servidores

Análisis y ajuste de parámetros de hardware y software para maximizar la eficiencia de servidores Linux y Windows. Se realiza tuning de CPU, memoria, disco y red, configuración de servicios críticos, cacheo, balanceo de procesos y monitoreo de recursos. Se aplican buenas prácticas para reducir tiempos de respuesta, evitar cuellos de botella y garantizar la estabilidad bajo cargas intensivas. Se utilizan herramientas como top, htop, iostat, vmstat y analítica de logs de rendimiento para decisiones basadas en datos.

Mantenimiento de firewalls

Configuración, actualización y monitoreo de firewalls físicos y virtuales (iptables, pfSense, UFW, firewalld). Se definen reglas de acceso, segmentación de redes, protección contra ataques externos e internos, y auditoría de tráfico. Se realizan revisiones periódicas, actualización de firmware, generación de reportes de eventos y respuesta ante incidentes para mantener la seguridad y disponibilidad de los sistemas. Se establecen también listas blancas, negras y zonas de seguridad por niveles.

Despliegue de aplicaciones

Instalación, configuración y puesta en marcha de aplicaciones empresariales, educativas y de desarrollo en servidores Linux y Windows. Se gestionan dependencias, bases de datos, servicios web y seguridad. Se automatizan procesos de despliegue con herramientas como Docker, Ansible y scripts personalizados, asegurando funcionamiento estable y seguro. Se crean entornos de staging, integración continua y se documenta el procedimiento completo para reproducibilidad y soporte técnico.

Gestión de bases de datos

Administración integral de bases de datos MySQL, PostgreSQL, MariaDB y otras. Incluye instalación, configuración, respaldo, restauración, optimización de consultas, gestión de usuarios y permisos, y monitoreo de rendimiento. Se aplican políticas de seguridad y se automatizan tareas para garantizar la integridad y disponibilidad de la información. Además, se configuran réplicas, particionamiento, triggers, procedimientos almacenados y se analizan cuellos de botella en ejecución de sentencias SQL.

Configuración de balanceadores de carga

Implementación y ajuste de balanceadores de carga como HAProxy, Nginx y LVS para distribuir el tráfico entre múltiples servidores. Se configuran reglas de balanceo, chequeos de salud, failover y escalabilidad, mejorando la disponibilidad y el rendimiento de servicios web, aplicaciones y bases de datos. También se establecen métricas de control, integración con DNS dinámico, políticas de redirección y técnicas de persistencia de sesión para mantener la experiencia del usuario.

Auditorías de seguridad

Evaluación exhaustiva de sistemas y redes para identificar vulnerabilidades, malas configuraciones y riesgos potenciales. Se realizan análisis de logs, escaneo de puertos, revisión de políticas de acceso y simulación de ataques. Se entregan informes detallados y recomendaciones para fortalecer la seguridad y cumplir normativas. También se realiza seguimiento post-auditoría para verificar la aplicación de correcciones y se ofrecen capacitaciones para mejorar prácticas internas.

Soporte en migraciones

Planificación y ejecución de migraciones de sistemas operativos, aplicaciones, bases de datos y servicios entre servidores físicos, virtuales o en la nube. Se minimizan tiempos de inactividad, se asegura la integridad de los datos y se realiza documentación de todo el proceso para facilitar futuras actualizaciones. Se implementan pruebas piloto, backups previos, validaciones post-migración y planes de contingencia ante errores.

Monitoreo de logs

Configuración de sistemas de registro y análisis de logs (Syslog, Logrotate, ELK Stack) para detectar incidentes, anomalías y optimizar procesos. Se implementan alertas automáticas, reportes periódicos y políticas de retención, facilitando la auditoría y la respuesta rápida ante problemas. También se integran filtros por severidad, correlación de eventos y dashboards visuales para facilitar el análisis diario.

Diseño de redes empresariales

Planificación y desarrollo de la arquitectura de redes empresariales seguras y eficientes. Incluye segmentación de redes, configuración de switches y routers, enlaces redundantes, VLANs, QoS y seguridad perimetral. Se realiza un relevamiento de necesidades, simulaciones de topología y se entrega documentación técnica completa. El diseño contempla escalabilidad, resiliencia y compatibilidad con servicios existentes.

Gestión de dominios y DNS

Administración de dominios y configuración de registros DNS para asegurar la correcta resolución de nombres. Se gestionan zonas directas e inversas, TTLs, redirecciones, subdominios, y validaciones SPF, DKIM y DMARC para correo electrónico. Se configuran servicios con Bind, Cloudflare o proveedores de hosting, asegurando disponibilidad global y protección contra secuestros de dominio.

Implementación de sistemas de correo electrónico

Configuración de servidores de correo, cuentas y políticas antispam para comunicación empresarial segura. Se implementan servidores como Postfix, Dovecot o Microsoft Exchange, filtros de virus, listas negras, cifrado SSL/TLS y autenticación segura. Se integran clientes web como Roundcube u Outlook y se documenta el uso para los usuarios finales.

Capacitación a usuarios finales

Formación y asesoría a usuarios sobre el uso seguro y eficiente de sistemas y herramientas tecnológicas. Se diseñan manuales personalizados, sesiones prácticas, talleres remotos o presenciales, y evaluaciones para verificar el aprendizaje. Se abordan temas como navegación segura, uso de software corporativo, prevención de malware y buenas prácticas en el uso del correo electrónico y redes internas.

Configuración de sistemas de monitoreo

Implementación de herramientas para supervisar el estado y rendimiento de servidores y servicios. Se configuran soluciones como Zabbix, Nagios, Prometheus y Grafana para monitoreo activo y reactivo, con paneles visuales, alertas por correo o Telegram, y generación automática de informes. Se integran métricas de red, uso de recursos, disponibilidad de servicios y logs del sistema.

Integración de servicios en la nube

Conexión y administración de servicios cloud como AWS, Azure, Google Cloud, entre otros. Se realiza el despliegue de servidores virtuales, almacenamiento escalable, redes privadas virtuales, gestión de identidades y automatización mediante Terraform o scripts. Se asegura la integración con sistemas locales, la continuidad del negocio y se configuran respaldos y seguridad en entornos híbridos.

Gestión de incidentes críticos

Respuesta rápida y efectiva ante incidentes graves que afectan la operación de sistemas y redes. Se realiza diagnóstico inmediato, contención, análisis de causas y restauración del servicio. Se elaboran planes de recuperación, protocolos de escalamiento y documentación de cada incidente. Se incluyen análisis forense, monitoreo posterior y recomendaciones para evitar recurrencias.

Soporte en proyectos de TI

Asistencia técnica y operativa en la ejecución y desarrollo de proyectos tecnológicos. Se colabora en el diseño de soluciones, adquisición de equipos, pruebas piloto, validación de requerimientos, migraciones, y documentación. También se apoya en la planificación, cronogramas, análisis de riesgos y comunicación con el equipo técnico y usuarios finales.

Configuración de políticas de acceso remoto

Definición y aplicación de reglas para el acceso seguro y controlado a sistemas desde ubicaciones externas. Se implementan conexiones por VPN, autenticación de dos factores, firewalls adaptativos, túneles cifrados y segmentación de permisos según el rol. Se monitorean accesos, se registran sesiones y se aplican políticas de horario o ubicación geográfica para mayor control.
